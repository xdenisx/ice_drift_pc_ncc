# Geotiff files mosaicing

Usage: 

```
This script creates a mosaic from a set of individual geotiff files within a defined time lag (NUMBER_OF_DAYS) from the latest one

python merge_s1_geotiff_files_dates.py /PATH/TO/S1/GEOTIFF/FILES FILE_MASK[S1] /OUTPUT/PATH/FOR/MOSAIC NUMBER_OF_DAYS[1,2,3,...] 

```
